# grafana

all about grafana

# Install Grafana on macOS

Commands to install grafana

```
brew update
brew install grafana
```

# Start Grafana

`brew services start grafana`

# Access Grafana

The service will be running on the url
`http://localhost:3000/login`

- Credentials below
  _username_: `admin`
  _password_: `admin`
